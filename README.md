# `hello-parcel-app`

## Setup

### First time

```bash
$> npm run setup
```

### Dependencies

```bash
$> npm install
```

## Development

### Build: `dist/`

```bash
$> npm run dist
```

### R&D

#### serve

```bash
$> npm start
```

Will host it in http://localhost:1234

#### browser

```bash
$> npm run browser
```

Opens a new tab on http://localhost:1234
